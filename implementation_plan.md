# Implementation Plan — Hide Bottom Lock & Vercel URL Clarification

This plan details the layout simplification of the dashboard for free plan users and explains the difference between the various Vercel deployment URLs.

---

## 1. Vercel URL Clarification (User Question #2)

Out of the three URLs showing in your Vercel dashboard:
1. **`https://heartmind-ai.vercel.app/`**: This is your **original, official Production URL**. It always points to the latest version of the `main` branch. This is the link you should share with users.
2. **`heartmind-ai-eby3.vercel.app` & `heartmind-ai-i67h.vercel.app`**: These are **Preview Deployment URLs** automatically generated by Vercel for each individual commit. They are snapshots of older code versions and can be ignored.
